# j - AI Companion

개인용 한국어 companion 채팅 앱 1차 프로토타입입니다.

## 현재 포함된 기능

- iPhone Safari에서 열 수 있는 웹앱 화면
- j 캐릭터 설정 반영
  - 여성
  - 20대 중반
  - 다정하고 장난기 있지만 과하지 않은 말투
  - 관계 시작점: 친해진 친구
  - 관계 발전: 친구 → 서로 신경 쓰는 사이 → 썸 → 연인 → 안정적인 연인
  - 질투/서운함 강도: 중
- 대화 내용, 친밀도, 관계 단계가 브라우저에 저장됨
- 답장이 오래 늦으면 j가 살짝 서운함을 표현함
